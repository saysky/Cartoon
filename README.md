## 项目说明
基于 springboot+mybatis+thymeleaf实现的动漫网站。<br/>
优势：能发送弹幕、推荐使用协同过滤

## 功能介绍
- 包含游客、用户和管理员三种角色
- 游客可以只能看视频
- 用户可以发布视频、发送弹幕、留言、点赞等
- 管理员可以审核视频，视频管理、分类管理、标签管理、留言管理、用户管理等等
